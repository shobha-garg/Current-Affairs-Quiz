# 📰 Current Affairs Assessment Platform

An interactive web-based assessment platform designed to help aspirants practice **Banking, Economy, RBI, and Current Affairs** through timed quizzes with real-time score tracking and performance evaluation.

---

## 📌 Overview

The Current Affairs Assessment Platform is a full-stack web application that enables users to attempt current affairs quizzes in an interactive environment. The platform maintains user progress, calculates scores automatically, and provides a seamless experience through a responsive web interface.

The project was developed to simulate the experience of online assessment portals used for competitive examinations while demonstrating backend development using Flask and SQLAlchemy.

---

## ✨ Features

- 👤 User Registration & Login
- 📝 Interactive Multiple Choice Quizzes
- 📊 Real-Time Score Calculation
- 💾 Persistent Database Storage
- 📈 Performance Tracking
- 📱 Responsive User Interface
- ⚡ Fast and Lightweight Flask Backend
